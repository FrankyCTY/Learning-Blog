Firestore Introduction

What is Firestore?

Cloud firestore is a document database that stores the data in a tree like structure and everything is placed into documents and collections.

Document

It can be thought of a javascript object that consisted of key value pairs (fields), but with more data types that we can use in it.

Restriction:

Document can not point to other document, but it can point to subcollection.

Gray box is collection, blue box is document

Collection

Collection of documents, it can only contain document.
