Redux Thunk

‚≠êRedux Thunk is a middleware that we used to intercept the redux action before they get to the reducers.

It only intercept the action that return a function and will generate a dispatch as the first argument for the returning function. (Dispatch is used to dispatch action to the reducer)

It allows us to wrap a action flow, such as the asynchronous operation to fetch data.
