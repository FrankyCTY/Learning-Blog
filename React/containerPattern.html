Container Pattern

Before using container pattern

⭐ In the ShopPage Component, the operation to enhance a specific component (With HOC) doesn't relate to the ShopPage Component is exposed in the component js, which makes the code harder to read and manage.

What is not relating to the ShopPage component here?

  • Using Higher Order Component - WithSpinner for ApplicationOverview + ApplicationDetailPage
  • Getting the redux state "isApplicationsLoaded" for the above two components

With container pattern

⭐We create a container.jsx for the target component that compose a new component with the HOCs.

Compose start wrapping the target component with the HOC from the right. (WithSpinner - it returns a component)
